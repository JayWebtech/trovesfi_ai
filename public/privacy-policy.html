Privacy Policy for Troves.fi AI Assistant

Last Updated: October 12, 2025

Introduction

This Privacy Policy describes how Troves.fi AI Assistant ("we", "our", or "the Bot") collects, uses, and protects your information when you use our chatbot services via Telegram and WhatsApp.

Information We Collect

1. User Messages

  • What: Text messages you send to the bot
  • Why: To process your queries and provide responses about Troves.fi yield strategies
  • Retention: Conversation history is kept temporarily in memory for context (last 12 messages)

2. User Identifiers

  • What: Your Telegram User ID or WhatsApp phone number
  • Why: To maintain conversation context and provide personalized responses
  • Storage: Stored temporarily in server memory, not in permanent databases

3. Wallet Addresses (Optional)

  • What: Starknet wallet addresses you provide when checking balances
  • Why: To query blockchain data and show your vault balances
  • Storage: Not stored. Only used in real-time for blockchain queries

How We Use Your Information

We use collected information to:

  • Process queries about Troves.fi yield strategies, APY, TVL, and investments
  • Fetch real-time blockchain data from Starknet contracts
  • Maintain conversation context for better responses
  • Debug issues and enhance bot functionality
  • Calculate investment projections based on current strategy data

Information We DO NOT Collect

We do NOT collect:

  • Personal identification information (name, email, etc.)
  • Financial information or payment details
  • Private keys or wallet credentials
  • Location data
  • Contact lists
  • Any sensitive personal data

Data Storage and Security

Storage

  • Conversation History: Stored temporarily in server memory (RAM only)
  • No Permanent Database: We do not maintain a persistent database of user messages
  • Session-Based: Data is cleared when server restarts or sessions expire
  • Auto-Cleanup: Conversation history limited to last 12 messages per user

Security Measures

  • All API communications use HTTPS encryption
  • Server hosted on secure infrastructure (Railway)
  • No storing of sensitive information
  • Blockchain queries are read-only operations
  • Access tokens and API keys stored securely in environment variables

Third-Party Services

1. Anthropic AI (Claude)

  • Purpose: Process natural language queries using AI
  • Data Shared: Your messages (without personal identifiers)
  • Privacy Policy: anthropic.com/privacy

2. Starknet Blockchain

  • Purpose: Query public blockchain data (balances, TVL, yields)
  • Data Shared: Wallet addresses you provide (public blockchain data)
  • Nature: All blockchain data is publicly available

3. Troves.fi API

  • Purpose: Fetch current strategy data (APY, TVL, contract addresses)
  • Data Shared: None. Read-only API calls
  • Website: app.troves.fi

4. Telegram/WhatsApp

  • Purpose: Message delivery platforms
  • Data Shared: Messages sent through their platforms
  • Policies:
    • Telegram: telegram.org/privacy
    • WhatsApp: whatsapp.com/legal/privacy-policy

Your Rights

You Have the Right To:

  • Access: Request information about what data we have about you
  • Deletion: Request deletion of your conversation history
  • Opt-Out: Stop using the bot at any time
  • Data Portability: Request a copy of your conversation history

How to Exercise Your Rights:

  • Send a message to the bot: "Delete my data" or "Clear my history"
  • Stop using the service at any time by not sending messages

Data Retention

  • Conversation History: Maximum 12 messages per user, cleared on server restart
  • Logs: Server logs rotated every 7 days
  • No Long-Term Storage: We do not maintain long-term user databases
